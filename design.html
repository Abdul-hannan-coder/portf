Azure Deployment for Decentraland Metaverse with integrated RASA chatbots

This project involved a comprehensive deployment of a RASA chatbot within the Decentraland metaverse, hosted on Microsoft Azure, to enhance user interaction and engagement.

Project Visuals

Project Objectives

  • Deploy a scalable and reliable RASA chatbot on Azure cloud infrastructure.
  • Integrate the chatbot seamlessly into the Decentraland metaverse environment.
  • Ensure high availability and low latency for real-time user interactions.
  • Implement CI/CD pipelines for automated testing and deployment.

Detailed Description

We worked with Mian on the deployment of a Decentraland Metaverse integrated with Rasa chatbots, and his work met all expectations. The project required setting up a robust backend on Azure, including virtual machines, networking, and security groups. Mian configured Nginx as a reverse proxy and used Docker Compose to manage the containerized application services. GitHub Actions were implemented to create a full CI/CD pipeline, automating the build, test, and deployment process, which significantly streamlined our development workflow and reduced manual errors. His expertise in both RASA and cloud infrastructure was pivotal to the project's success.
